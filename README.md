# Plain AI - RSI 진단 분석 채팅

Gemini AI를 활용한 웹 기반 채팅 인터페이스

## 설치 및 실행

```bash
# 1. 저장소 클론
git clone https://github.com/Plain-Insight/plain_ai.git
cd plain_ai

# 2. Node.js 의존성 설치
npm install

# 3. 환경변수 설정
cp .env.example .env
# .env 파일을 열어서 GOOGLE_API_KEY=your_api_key_here 입력

# 4. 서버 실행
npm start
```

## 접속

브라우저에서 <http://localhost:3000> 접속

## 요구사항

- Node.js 16+
- Google Gemini API 키
